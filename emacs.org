* straight.el Setup

straight.el is a package manager for emacs that is

 - compatible with use-package,
 - allows to pin down the version of each package it installs, and
 - downloads the source code of each package it installs and compiles it.


 The following code snippet downloads and installs straight.el. In case this does not work one can clone the straight.el repo from github into ~/.emacs.d/straight/repos.

#+begin_src emacs-lisp

  (defvar bootstrap-version)
  (let ((bootstrap-file
         (expand-file-name
  	"straight/repos/straight.el/bootstrap.el"
  	(or (bound-and-true-p straight-base-dir)
  	    user-emacs-directory)))
        (bootstrap-version 7))
    (unless (file-exists-p bootstrap-file)
      (with-current-buffer
  	(url-retrieve-synchronously
  	 "https://radian-software.github.io/straight.el/install.el"
  	 'silent 'inhibit-cookies)
        (goto-char (point-max))
        (eval-print-last-sexp)))
    (load bootstrap-file nil 'nomessage))

  (straight-use-package 'use-package)

#+end_src

** Font Setup

Currently I am using /JetBrains Mono/ and /Source Serif 4/ fonts. These fonts are located in the fonts subfolder in case you are on windows and did not install them. Otherwise the nix flake will install them automatically once you include the
additional-emacs-packages in you packages.

The mixed pitch package tries to set variable-pitch-fonts for where it is appropriate. 

#+begin_src emacs-lisp

  (defvar pl/default-font-size 140)
  (defvar pl/default-font "JetBrains Mono")
  (defvar pl/variable-pitch-font "Source Serif 4")
  (defvar pl/variable-pitch-font-size 140)
  (defvar pl/fixed-pitch-font "JetBrains Mono")
  (defvar pl/fixed-pitch-font-size 140)

  (message "Setting faces!")

  (when (member pl/default-font (font-family-list))
    (set-face-attribute 'default nil
  		      :font pl/default-font
  		      :height pl/default-font-size))

  (use-package mixed-pitch
    :straight t
    :config
    (when (member pl/variable-pitch-font (font-family-list))
      (set-face-attribute 'variable-pitch nil
  			:family pl/variable-pitch-font
  			:height pl/variable-pitch-font-size)))

  (when (member pl/fixed-pitch-font (font-family-list))
    (set-face-attribute 'fixed-pitch nil
  		      :font pl/fixed-pitch-font
  		      :height pl/fixed-pitch-font-size))

** Setup transparency

Working in a slightly transparent frame is neet! So let's configure it. Unfortunately, it does work a little different on windows, since the variable alpha-background is not supported there. 

#+begin_src emacs-lisp

  (defun pl/set-transparency ()
  "Set the transparency according to the platform."
  (cond
   ((eq system-type 'windows-nt)
    (set-frame-parameter (selected-frame) 'alpha '(95 . 100))
    (add-to-list 'default-frame-alist '(alpha . (95 . 100)))
    (message "Running on Windows"))
   ((eq system-type 'gnu/linux)
    (set-frame-parameter nil 'alpha-background 95)
    (add-to-list 'default-frame-alist '(alpha-background . 95))
    (message "Running on GNU/Linux"))
   (t
    (message "Unknown operating system"))))

#+end_src

** Putting things together

We have to make sure that a frame is set up when we are in running in a daemon.

#+begin_src emacs-lisp

  (defun pl/setup-gui ()
    "Setup the gui."
    (pl/set-transparency))

  (if (daemonp)
      (add-hook 'after-make-frame-functions
  	      (lambda (frame)
  		;; (setq doom-modeline-icon t)
  		(with-selected-frame frame
  		  (pl/setup-gui)
  		  )))
    (pl/setup-gui))

#+end_src

* Advanced UI configuration

I use /doom-modeline/ and /doom-themes/ since these two are nicely integrated and the colors match a wide a range of needs and packages.

In addition I load the /all-the-icons/ and /nerd-icons/ to have them as well.

#+begin_src emacs-lisp
  
  (use-package all-the-icons
    :straight t
    :if (display-graphic-p))

  (use-package nerd-icons
    :straight t)

  (use-package doom-modeline
    :straight t
    :after all-the-icons nerd-icons
    :init (doom-modeline-mode 1))

  (use-package doom-themes
    :straight t
    :custom
    (doom-themes-enabe bold t)    ; if nil, bold is universally disabled
    (doom-themes enable-italic t) ; if nil, italics are universally disabled

    ;; use "doom-atom" for less minimal icon theme
    (doom-themes-treemacs-theme "doom-atom")

    :config
    (load-theme 'doom-one t)

    ;; Enable flashing mode-line on errors
    (doom-themes-visual-bell-config)
    
    ;; Enable custom neotree theme (nerd-icons must be installed)
    (doom-themes-neotree-config)
    
    ;; Corrects (and improves) org-mode's native fontification
    (doom-themes-org-config))

#+end_src
